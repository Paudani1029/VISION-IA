Programas necesarios (instalar en orden)
1. Node.js
Ve a nodejs.org y descarga la versión LTS
Instálalo con siguiente, siguiente, finalizar

Verifica que todo quedó instalado abriendo la terminal y corriendo:
node --version
python --version
git --version

Descargar el proyecto
git clone https://github.com/Paudani1029/VISION-IA.git
cd VISION-IA

Configurar el backend
cd vision-ia-backend
pip install -r requirements.txt

(La api_key que hice ya se le acabo la prueba gratuita, asi que posiblemente no te funcione y tengas que crear una)
En el archivo .env dentro de la carpeta vision-ia-backend con este contenido:
GEMINI_API_KEY=tu_api_key_aqui

Para obtener la API key gratis (La de Gemini):

Ve a aistudio.google.com
Inicia sesión con tu cuenta de Google
Clic en "Get API Key" → "Create API key"
Copia la key y pégala en el .env

Configurar el frontend
cd ../vision-ia-frontend
npm install --legacy-peer-deps

(Esto esta listo)
Crear un archivo llamado .env dentro de la carpeta vision-ia-frontend con este contenido:
VITE_API_URL=http://localhost:8000

Levantar el proyecto
Necesitas dos terminales abiertas al mismo tiempo:
Terminal 1 — Backend:
cd vision-ia-backend
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000

Terminal 2 — Frontend:
cd vision-ia-frontend
npm run dev

IDEA:
Podríamos hacer una especie de evaluación de base de datos. 
Que la IA analice los datos que ha recopilado de su usuario, y deduzca ciertas cosas:
1) Recoge:
- Lugares más transitados
- Personas con mayor interacción
- Amenazas detectadas (caras enojadas que el usuario no notaría porque no las interactúa, esquinas con mucho tráfico, perros agresivos, calles peligrosas -usuario no percibe, porque ocurre a lo lejos de él) 
    --> guardarlas en una memoria especial
2) En base a ello:
- Obstáculos más frecuentes / más probables
- Generar nuevas advertencias: "cuidado con esa casa, perro peligroso"; "te sugiero que vayas por otra calle"
- Si usuario retroalimenta, la IA aprende que esa situación requiere menos outputs de su parte. 
    Ejemplo: "no, ese perro es amigo mío, no me hará nada" --> no volverá a advertir sobre esa casa específica.
